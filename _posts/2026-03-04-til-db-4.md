## 학습내용
- 테이블의 생성과 확인, 삭제

### 1. 테이블 생성
- MyDatabase 데이터베이스 안, MyTable 테이블 생성
```sql
CREATE TABLE MyTable(
    id INT AUTO_INCREMENT PRIMARY KEY, -- 고유 ID
    name VARCHAR(50), -- 문자 입력
    number INT --숫자 입력
); -- MyTable이라는 이름의 테이블 및 컬럼 생성
CREATE TABLE IF NOT EXISTS MyTable(); -- MyTable이라는 이름의 테이블이 없을 경우 생성
```

#### 그 외 데이터 타입
- 정수: INT
- 소수: FLOAT, DOUBLE (적당히) / DECIMAL (정확하게)
- 문자열: VARCHAR, TEXT (긴 글)
- 날짜 (및 시간): DATE, DATETIME, TIMESTAMP
- 상태값: BOOLEAN, ENUM

### 2. 테이블 확인
- 테이블 목록 및 구조 확인
```sql
SHOW TABLES; -- 현재 데이터베이스의 테이블 목록 보기
DESC MyTable; -- MyTable 테이블 구조 확인
```

### 3. 테이블 삭제
- MyTable 테이블 삭제
```sql
DROP TABLE MyTable; -- MyTable 테이블 삭제
DROP TABLE IF EXISTS MyTable; -- MyTable 테이블이 있을 경우 삭제
```
