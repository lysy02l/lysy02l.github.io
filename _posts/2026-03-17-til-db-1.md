## 학습내용
- MySQL과 Python의 연동.
- Python 환경으로 데이터베이스 입력과 수정.

``` python
import mysql.connector  #MySQL과 연결할 모듈 불러옴 

conn = mysql.connector.connect(  #connect 함수 호출, 접속할 MySQL 정보
    host="localhost", #접속할 MySQL 서버의 주소
    user="root", #데이터베이스에 접속하는 사용자
    password="비밀!!", #해당 사용자 계정 비밀번호(!대놓고 적어두는 건 보안상 위험!)
    database="python_db" #사용할 데이터베이스 이름
)  

cursor = conn.cursor() #데이터베이스에 명령을 보내기 위한 객체(작업 도구) 지정

cursor.execute("SQL 명령문") #SQL 명령문 실행
conn.commit() #데이터베이스에 저장 확정

cursor.execute("SELECT * FROM users")
print(cursor.fetchall()) #Select된 결과 행 전체 출력

cursor.close() 
conn.close() #작업 도구 및 접속 종료
```
