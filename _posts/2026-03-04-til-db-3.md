## 학습내용
- 데이터베이스의 생성과 확인, 삭제

### 1. 데이터베이스 생성
- MyDatabase 데이터베이스 생성
```sql
CREATE DATABASE MyDatabase; -- MyDatabase라는 이름의 데이터베이스 생성
CREATE DATABASE IF NOT EXISTS MyDatabase; -- MyDatabase라는 이름의 데이터베이스가 없을 경우 생성
--
USE MyDatabase; -- 작업을 위해 MyDatabase 데이터베이스에 접속
```

### 2. 데이터베이스 확인
- 데이터베이스 목록 확인
```sql
SHOW DATABASES; -- 현재 존재하는 데이터베이스 목록 보기
```

### 3. 데이터베이스 삭제
- MyDatabase 데이터베이스 삭제
```sql
DROP DATABASE MyDatabase; -- MyDatabase라는 이름의 데이터베이스 삭제
DROP DATABASE IF EXISTS MyDatabase; -- MyDatabase라는 이름의 데이터베이스가 있을 경우 삭제
```
