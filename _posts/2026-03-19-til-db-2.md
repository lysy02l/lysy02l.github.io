```sql
create table MyCharacter (
    name varchar(20) not null,
    level int not null,
    ATK int not null,
    DEF int not null,
    Crit_Rate float,
    Crit_Damage float
);
```

&nbsp;저번에 Python과 MySQL을 연결해 봤으니, 안 쓰기에는 뭐 해서... 저번 양식 그대로, 스리슬쩍 Python도 꼽사리 한 번 끼워봤습니다.
```python
import mysql.connector 

conn = mysql.connector.connect( 
    host="localhost",
    user="root",
    password="비밀!!", 
    database="game_db" 
)  

cursor = conn.cursor() 

cursor.execute("insert into MyCharacter (name, level, ATK, DEF, Crit_Rate, Crit_Damage) values ('Flins', 90, 2486, 935, 61.3, 236.9)") 
conn.commit()

cursor.execute("SELECT * FROM MyCharacter")
print(cursor.fetchall()) 

cursor.close() 
conn.close() 
```
