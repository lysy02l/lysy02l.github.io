## 학습내용
- 데이터의 생성, 조회, 수정, 삭제

### 1. 데이터 생성
- MyDatabase 데이터베이스 안, MyTable 테이블 안, 데이터 추가
```sql
INSERT INTO MyTable (name, number) VALUES ('data_1', 10); -- 데이터 추가
--
INSERT INTO MyTable (name, number) VALUES 
('data_2', 20),
('data_3', 30); -- 여러 데이터 한 번에 추가
```

### 2. 데이터 조회
- MyTable 테이블 데이터 조회
```sql
SELECT * FROM MyTable; -- 전체 조회
SELECT name FROM MyTable; -- 특정 컬럼만 조회
```

### 3. 데이터 수정
- MyTable 테이블 데이터 수정
```sql
UPDATE MyTable
SET number = 40
WHERE name = 'data_1'; -- 특정 데이터 수정(이름이 data_1인 행의 number를 40으로 수정)
-- WHERE(조건) 없으면 모든 데이터의 number가 40이 됨
```

### 3. 데이터 삭제
- MyTable 테이블 데이터 삭제
```sql
DELETE FROM MyTable
WHERE name = 'data_2'; -- 특정 데이터 삭제
-- WHERE(조건) 없으면 모든 데이터 날아감
```
