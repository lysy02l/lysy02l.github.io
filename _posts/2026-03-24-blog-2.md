## 1. 블로그 미리보기 (로컬 브라우저 실행)
- text.bat

```bash
@echo off

for /f "tokens=5" %%a in ('netstat -aon ^| find ":4000" ^| find "LISTENING"') do (
    taskkill /f /pid %%a >nul 2>&1
) #실행 중인, 4000번 포트 쓰는 서버 죽여버리기

cd C:\ #폴더 위치 입력
start "" cmd /c "bundle exec jekyll serve" #서버 실행

:wait
curl -s http://127.0.0.1:4000/ >nul
if errorlevel 1 (
    timeout /t 1 >nul
    goto wait
) #서버 켜질 때까지 기다렸다가..

start http://127.0.0.1:4000/ #브라우저 실행
```

## 2. 블로그 업로드

```bash
@echo off

cd C:\ #폴더 위치 입력

set /p msg=text: #업로드 제목 입력 

git add .

pause
```
